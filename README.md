# JSX Profile Card

This project converts a basic HTML profile card into JSX so it can be used inside a React component.

I created a `UserProfile` component that receives `name` and `email` as props and displays them in the profile card. Instead of hardcoding the values, they are passed from `App.jsx`.

I also added a few test cases, including some edge cases like empty values and a very long name, to make sure the component still renders correctly.

## Running the Project

1. Clone or download the project and open it in VS Code (or any editor you prefer)
2. Run `npm install`
3. Run `npm run dev`
4. Open the local server link that appears in the terminal
