# JSX Profile Card – AD311

This is my assignment for AD311 – Intermediate Development 1. The goal was to take a basic HTML profile card and convert it into JSX, then use it inside a real React component.

I built a `UserProfile` component that takes `name` and `email` as props. I also tested it with a few edge cases like empty values and really long names to see how it handled them.

## Running the Project

1. Clone or download the project and open it in VS Code or your preferred IDE
2. Run `npm install`
3. Run `npm run dev`
4. Open the local server link shown in the terminal
